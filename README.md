# Siksha
IP validator
1.Built on angular-6
  Three component:
	1. Login
	2. Basic
	3. Premium


2.Url : 
  http://localhost:4200
  http://localhost:4200/Basic - for basic user type (Can take upto 5 IP addressess)
  http://localhost:4200/Premium - for Premium user type (Can take upto 10 IP addressess)



3. To Run :
   From angular CLI
	type : ng serve
